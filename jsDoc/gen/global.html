scrollTo(value, withAnimation, withSticky)

scrollTo 공통 해당 위치로 스크롤

Parameters:
Name Type Description
value number | HTMLElement | selector | jQuery

scroll할 숫자 또는 htmlElement 또는 셀렉터스트링 또는 jQuery
withAnimation boolean

기본: true, 애니메이션 여부
withSticky boolean

기본: true, 해더등의 스티키 상황인지에 따라 top offset변경 됨
Example
scrollTo(100);
scrollTo(document.getElementById('id');
scrollTo('.some-class');
scrollTo($('.some-class'));
