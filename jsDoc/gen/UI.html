Class: UI

UI

모든 UI는 UI클래스를 상속받는다.

new UI(name, el)

UI클래스를 상속받은 UI는 super(name, el)을 해줘야한다.

Parameters:
Name Type Description
name string

UI이름을 정의한다(example: tooltip)
el HTMLElement

UI요소 한개에 해당하는 요소를 넣는다.(ui-xxxx클래스가 붙는 ui요소)

Methods

getRandomStr()

최대 6자리 랜덤 숫자

Returns:
Type
number

parseOptions(options)

옵션은 모두 data-xxx 형태로 넘어오게 되고 해당 값의 유형에 맞게 변환해준다. $('selector').data() 하게되면 해당 data-요소가 모두 들어오게 된다.

Parameters:
Name Type Description
options

예: data-no="1" data-type="type1" { no: 1, type: 'type1' }
Returns:
Type
object

setEventId()

ui의 이벤트ID 생성 ex) uiName-0000
